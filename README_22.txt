== PROJECT MANAGER ===================================================
-- Gruppe 22

-- Overblik ----------------------------------------------------------
Applikationen: src > projectmanager > app > ProjectManagerApp.java
Cmd line interface: src > projectmanager > ui > ProjectManagerUI.java
Systematiske tests: test > projectmanager > app

-- Instruktioner -----------------------------------------------------
Til at køre CLI'et får I brug for employee usernames. Vi har uploadet 
et "state" med (i pmapp.data), hvor der allerede er registreret 
employees og projekter. Dertil kan I bruge flg. brugernavne: 
	aid, qvqi, sjhd

Hvis I vil lave et nyt state kan I køre TestSaveObject.java (under 
tests). Men husk at tjekke de brugernavne, den printer ud blandt alle 
data, så I kan logge ind.

God fornøjelse!
